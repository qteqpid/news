## 高价值灵感

### 1. Agent Page Guard
- 来源信号：AI browser 攻击报道、agent 使用扩大、普通用户难判断网页 prompt injection。
- 金子类型：平台缺口、关键词供给缺口、高痛点低成熟度。
- 目标用户：使用 AI 浏览器、浏览器 agent、自动化插件的研究者、开发者、重度办公用户。
- 痛点：用户不知道当前网页是否包含隐藏指令、恶意表单、可疑跳转或会污染 agent 上下文的内容。
- MVP：浏览器扩展扫描页面文本和 DOM 可疑区域；标注“可能诱导 agent 执行”的片段；给出复制/总结/自动填写前的风险提示；本地规则优先，必要时接小模型。
- 变现：Freemium，个人 Pro 每月订阅；团队版提供策略模板和审计日志。
- 难度：中。
- 推荐指数：8.6/10。
- 为什么现在做：AI 浏览器和 agent 产品增加，但用户侧安全提示仍很薄，安全厂商产品通常过重。
- 风险：浏览器权限敏感；误报会影响体验；需要持续更新攻击模式。
- 下一步调研：搜索 Chrome Web Store 里 prompt injection / AI browser security 扩展数量与评分；访谈 5 位 agent 重度用户。

### 2. Privacy Alias Auditor
- 来源信号：Hide My Email bug 与 WhatsApp usernames 争议说明“身份遮罩”用户并不真正知道自己暴露了什么。
- 金子类型：高需求低信任、平台功能缺口。
- 目标用户：重视隐私的 iOS/Mac 用户、自由职业者、经常注册海外服务的人。
- 痛点：用户创建了很多 alias/email/username，但不知道哪些已暴露、哪些重复使用、哪些绑定了重要账号。
- MVP：本地导入 alias 列表；标注重复、弱别名、真实姓名泄露风险；生成换绑清单；支持 CSV/1Password/Apple 邮箱规则手动导入。
- 变现：一次性买断 + 高级同步订阅。
- 难度：中。
- 推荐指数：7.9/10。
- 为什么现在做：隐私功能越来越普及，但“管理隐私别名”仍像密码管理器早期一样混乱。
- 风险：导入数据敏感；与系统级 API 集成可能受限。
- 下一步调研：查看 App Store 中 email alias manager、privacy audit、username checker 的搜索结果和评分。

### 3. Disc-to-Digital Collection Ledger
- 来源信号：主机游戏实体光盘减少，Xbox/PlayStation 讨论 disc-to-digital 和数字收藏迁移。
- 金子类型：情绪强需求、陈旧工具替代、国家/平台缺口。
- 目标用户：拥有实体游戏收藏的主机玩家、小型游戏店、二手游戏交易用户。
- 痛点：玩家不知道哪些实体盘已经数字化、哪些 DLC/账号权益绑定在哪个平台，二手交易记录也容易丢。
- MVP：扫描条形码建库；记录平台、版本、数字兑换状态、购买价格、借出/出售状态；导出保险/二手交易清单。
- 变现：买断制移动 app；收藏超过一定数量后付费解锁；小店铺版按年收费。
- 难度：低到中。
- 推荐指数：7.4/10。
- 为什么现在做：实体到数字的过渡期会制造强烈的“不确定资产”焦虑。
- 风险：游戏数据库覆盖率；条形码地区差异；需要处理用户贡献数据质量。
- 下一步调研：对比 CLZ Games、Gameye、PriceCharting 的评分、更新频率和本地化空白。

## 今日趋势信号
- AI 安全从企业后台走向普通浏览器和个人 agent 使用场景。
- 隐私功能越多，用户越需要“我到底暴露了什么”的可视化工具。
- 数字所有权转型让收藏、迁移、凭证记录类小工具有新入口。
